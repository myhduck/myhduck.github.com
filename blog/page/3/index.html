Mr. Data Converter

Jun 27th, 2012 | Comments

Mr. Data Converter

CSV형식의 파일 또는 TAB으로 구분되어 있는 파일의 내용을
여러가지 데이터 형식으로 변환할 수 있는 사이트를 알게되어서 소개할까 한다
변환 가능한 형식은 XML, JSON, PHP Array, HTML Table, MYSQL Insert 등등 여러형식으로 변환이 가능하다
사용방법은 간단하다 콤마 또는 탭으로 구분되어있는 파일내용을 복사 붇여넣기만 하면 끝!!

사이트는 여기에서 확인가능하다

엑셀파일의 내용을 데이터베이스에 밀어넣거나 xml 혹은 json 형태로 변환할 일이 가끔있는데 유용하게 쓰일듯 하다
